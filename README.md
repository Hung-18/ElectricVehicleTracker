Overview: 
In Vietnam, electric vehicles are becoming a trend and are encouraged by the government due to their convenience and environmental friendliness.
Project focus: 
Because electric vehicles are used, a system for repairing or monitoring vehicle status in real time is needed.
Key Technical implementation:  
This project focuses on the principles and architecture of software development.
This project use Clean Architecture to organize layer. SOLID to clean code and easy testing , maintain system.
Using Entity Framework Core to process data with SQL Server
